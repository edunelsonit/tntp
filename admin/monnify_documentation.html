TNTP Monnify Integration Guide

1. Getting Monnify Credentials

To enable Monnify payment processing:

  1. Sign up at https://app.monnify.com
  2. Go to Settings → API Keys
  3. Copy your:
    • API Key
    • Secret Key
    • Contract Code (from Dashboard)
  4. Update config/config.php: 
    define('MONNIFY_API_KEY', 'YOUR_API_KEY');
define('MONNIFY_SECRET_KEY', 'YOUR_SECRET_KEY');
define('MONNIFY_CONTRACT_CODE', 'YOUR_CONTRACT_CODE');
define('MONNIFY_BASE_URL', 'https://api.monnify.com'); // Change from sandbox
2. Testing Setup
Available Test Tools:
  • API Connection Test - Verify credentials and basic connectivity
  • Account Generator - Create virtual accounts for users
  • Webhook Test - Simulate payment transactions
  • Contract Discovery - Find correct contract code
  • Endpoint Explorer - Check available API endpoints
3. Account Generation Workflow

Step 1: Create a user via Admin → Users

Step 2: Navigate to Admin → Monnify

Step 3: Click "Generate All" or generate individual accounts

Result: Each user gets:

  • Virtual account number
  • Bank name (GTBank, Zenith, etc.)
  • Monnify reference ID
4. Testing Payments
Sandbox Testing:
  1. Go to Admin → Monnify → Generate Accounts
  2. Generate test accounts
  3. Go to Admin → Webhook Test
  4. Simulate payments (amounts will be recorded but not real)
  5. Check user payment status updates automatically
Production Payments:

Once live:

  • Users can transfer to their virtual accounts
  • Monnify sends webhook notifications
  • System automatically processes and updates records
5. Implementation Details
Database Fields:
  • virtual_account - The actual account number
  • bank_name - Bank providing the account
  • monnify_reference - Monnify's reference ID
Transaction Recording:

When payment is received:

  1. Monnify sends webhook with transaction details
  2. api/webhook.php validates signature
  3. Transaction recorded in database
  4. User's amount_paid updated
  5. payment_status recalculated (PAID/PARTIAL/DEFAULTING)
⚠️ Troubleshooting
Account Generation Fails:
  • Check API credentials in config/config.php
  • Verify contract code is correct
  • Ensure API key has "Reserve Account" permission
Webhook Not Received:
  • Ensure APP_BASE_URL is publicly accessible
  • Configure webhook URL in Monnify dashboard: APP_BASE_URL/api/webhook.php
  • Check webhook signature validation in api/webhook.php
Account Numbers Not Showing:
  • Check that getAllAvailableBanks is set to true
  • Verify contract has multiple bank partnerships
